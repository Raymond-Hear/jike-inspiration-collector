# 即刻灵感收集器

> 从即刻的深度内容中挖掘灵感，为你的Vibe Coding项目提供源源不断的选题。

## ✨ 是什么？

一个Chrome浏览器插件，帮助你在浏览即刻（https://web.okjike.com/）时，快速收集有价值的动态内容，整理成你的灵感知识库。

## 🎯 核心功能

### 1. 一键收集灵感
在即刻动态底部添加 **[💡 灵感]** 按钮，点击即可收集到本地。

### 2. 智能分类
将收集的灵感分为三类：
- **需求洞察**：发现产品需求和痛点
- **干货分享**：保存技巧和知识点
- **洞见思考**：记录深度观点

### 3. 本地管理
- 所有灵感存储在本地浏览器
- 支持按类型筛选、搜索
- 添加个人笔记和想法

### 4. 导出整理
- 导出为Markdown格式
- 导出为JSON格式（备份）
- 方便后续整理和回顾

## 🚀 安装方法

### 开发者模式安装

1. 下载本项目到本地
2. 打开Chrome浏览器，进入扩展管理页面：
   - 在地址栏输入：`chrome://extensions/`
   - 或者：菜单 → 更多工具 → 扩展程序
3. 开启右上角的**「开发者模式」**
4. 点击**「加载已解压的扩展程序」**
5. 选择包含 `manifest.json` 的文件夹（本项目根目录）
6. 完成！

## 💡 使用方法

### 收集灵感

1. 访问即刻网站（https://web.okjike.com/）
2. 浏览动态，找到感兴趣的内容
3. 点击帖子下方的 **[💡 灵感]** 按钮
4. 在弹窗中选择类型和添加笔记
5. 点击保存

### 查看灵感

1. 点击浏览器工具栏中的插件图标
2. 查看所有收集的灵感
3. 支持按类型筛选、搜索

### 导出灵感

1. 打开插件管理界面
2. 点击「导出」按钮
3. 选择导出格式（Markdown/JSON）
4. 保存到本地
