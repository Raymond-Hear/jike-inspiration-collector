# 更新日志

所有重要的项目变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.0.1] - 2025-01-08

### 新增
- ✨ 从即刻动态中收集灵感的核心功能
- ✨ 三种分类：需求洞察、干货分享、洞见思考
- ✨ 为每条灵感添加个人笔记
- ✨ 本地存储所有收集的灵感
- ✨ 按类型筛选灵感（全部/需求/干货/洞见）
- ✨ 搜索功能（搜索内容和笔记）
- ✨ 导出为 Markdown 格式
- ✨ 导出为 JSON 格式
- ✨ 一键清空所有灵感
- ✨ 空心黄色边框设计（#FFE411）
- ✨ 统计显示（总收集数、各类型数量）

### 技术细节
- 🎯 使用 Manifest V3
- 🎯 基于页面类型的智能注入策略
  - 列表页（首页/推荐/搜索）：注入所有按钮组
  - 详情页：只注入第一个按钮组（主动态），评论区不注入
- 🎯 自动导航模式：从列表页点击按钮自动跳转到详情页
- 🎯 Chrome 本地存储（chrome.storage.local）
- 🎯 响应式设计，适配即刻网站风格

### 已知问题
- ⚠️ 详情页按钮注入逻辑可能需要进一步优化
- ⚠️ 某些边缘情况下的内容提取可能不完整
