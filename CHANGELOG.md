## [0.0.1] - 2025-01-08

### ✨ 新增功能
- **灵感收集**：在即刻动态中添加 [💡 灵感] 按钮，一键收集有价值的内容
- **智能分类**：支持三种分类
  - 需求洞察 - 发现产品需求和痛点
  - 干货分享 - 保存技巧和知识点
  - 洞见思考 - 记录深度观点
- **个人笔记**：为每条灵感添加自己的想法和备注
- **本地管理**：所有灵感存储在浏览器本地，支持按类型筛选、关键词搜索
- **数据导出**：
  - 导出为 Markdown 格式（方便文档整理）
  - 导出为 JSON 格式（数据备份）
- **统计展示**：显示总收集数和各类型数量
- **黄色主题设计**：统一使用 #FFE411 黄色，空心边框风格

### ⚠️ 待优化功能
- **详情页按钮注入**：在某些情况下，按钮可能出现在评论区或主动态没有按钮
- **长文本提取**：超长动态（超过2000字符）的内容可能被截断
- **编辑功能**：暂不支持修改已收集灵感的分类和笔记
- **批量操作**：暂不支持批量删除、批量修改类型等操作

### 🛠️ 技术实现
- 使用 Manifest V3
- 基于页面类型的智能注入策略（列表页注入全部，详情页只注入主动态）
- Chrome 本地存储（chrome.storage.local）
- 自动导航模式（列表页点击自动跳转详情页）
